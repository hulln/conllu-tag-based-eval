# results/

Evaluation outputs and analysis reports.

- `runs/<run-id>/main/`: high-level outputs (eval summaries, model comparisons, QA report).
- `runs/<run-id>/diagnostics/`: verbose tagged metrics and error analyses.
- `archive/`: legacy output layouts from earlier iterations.
